1. клонируйте проект
	git clone git@github.com:Sidenser/test01.git
2. запустите сервер nodejs
	./test01/bin/www 
	or 
	node ./test01/bin/www
3. перейдите по адресу в браузере
	http://localhost:3000/
	
если нет nodejs то установите передя по ссылки ниже
http://nodejs.org/
